# 專案簡介

這是一個基於 Vue 的前端專案，目標是提供高效、易維護的元件化開發架構。

## 專案啟動方式

1. 安裝依賴：
   ```bash
   npm install
   ```
2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```
3. 打包專案：
   ```bash
   npm run build
   ```

## 主要技術棧
- Vue 3
- Vite
- JavaScript / TypeScript
- Pinia / Vuex（依專案實際情況）
- Vue Router
- 其他 UI 框架（如 Element Plus、Ant Design Vue 等）

## 目錄結構簡介

```
src/
  components/    # 可複用元件
  views/         # 各頁面
  router/        # 路由設定
  store/         # 狀態管理
  assets/        # 靜態資源
  utils/         # 工具方法
  ...
```
